## Grid-Flexbox-MediaQueries

A CSS Grid tutorial website by Brad Traversy.

## Code Changes

Changes were made to the original CSS and HTML code to demonstrate that there are many ways to produce the same UI for a given website.
The original CSS Grid code format was changed and where ever possible, it was replaced by Flexbox. More break points were introduced into the code using Media Queries.

## Tech Used

⚡️ CSS3
⚡️ CSS-Grid
⚡️ Flexbox
⚡️ HTML/5
